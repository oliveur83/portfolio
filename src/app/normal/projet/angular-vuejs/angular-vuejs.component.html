découverte des framework angular, react, vuejs

Problématique :


Au cours du premier semestre de mon Master 1, j'ai entrepris un projet passionnant axé sur la création d'une application d'administration scolaire intégrant les frameworks React, Vue, et Angular. L'objectif principal était de mettre en pratique les connaissances théoriques acquises dans le cadre du programme en explorant les fonctionnalités de ces frameworks populaires. En se penchant sur des concepts avancés tels que la gestion d'état, le routage, et les composants réutilisables, ce projet a été une opportunité essentielle pour consolider mes compétences en développement web et pour relever les défis spécifiques liés à la création d'une interface utilisateur dynamique et réactive.

fonctionnalités :


Mon projet d'application d'administration scolaire a été enrichi de fonctionnalités destinées à améliorer l'expérience utilisateur et à mettre en avant les aspects pratiques des frameworks React, Vue, et Angular. Un mode multifenêtres a été intégré pour permettre aux utilisateurs de naviguer facilement entre les différentes sections de l'application, illustrant ainsi la flexibilité et la modularité offertes par ces frameworks. De plus, des fonctionnalités de gestion d'étudiants, de notes, et de communication entre enseignants et étudiants ont été implémentées pour démontrer la puissance et la facilité d'utilisation de ces technologies

backend :


Concernant la liaison avec le backend, cela était le projet du semestre 1 réalisé avec un framework PHP.

Technologies Utilisées :


Frontend : angular,vuejs.
backend: php Base de Données : Mysql pour stocker les informations.
